##Project Description

This project is a Countries Search Web App built using JavaScript that fetches country data from an external API and displays it dynamically on the webpage.

When the page loads, all countries are displayed along with their flags, names, and population. Users can search for a specific country using the input field, and the app filters and shows matching results in real-time.


##Technologies Used

HTML5 – Structure (input field, results container, spinner) 

CSS3 – Styling (layout, flags, responsiveness) 

JavaScript (ES6) – DOM manipulation and logic

Fetch API – For making HTTP requests

REST API – Countries data API

##Features

🌍 Fetch and display all countries on page load 

🔍 Real-time search and filtering

🏳️ Displays country flag, name, and population

⏳ Loading spinner while fetching data

🔄 Dynamically updates UI without page reload

🧠 Efficient filtering using JavaScript

##How It Works
When the page loads:
API request is made to fetch country data
Spinner is displayed while loading
Data is stored and rendered on the screen
When the user types in the search box:
Input value is captured
Data is filtered based on country name
Matching results are displayed instantly
📷 Output

Each country shows:

🏳️ Flag image
🌐 Country name
